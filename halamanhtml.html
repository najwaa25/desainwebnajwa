Apa Itu HTML?

HTML (Hypertext Markup Language) adalah bahasa standar untuk membuat halaman web. HTML digunakan untuk membuat struktur dan konten dasar dari sebuah halaman web.

Materi Dasar HTML

  • Struktur Dasar HTML: Elemen-elemen dasar seperti <html>, <head>, dan <body>
  • Tag HTML: Penggunaan berbagai tag seperti <h1>, <p>, <a>, dan lainnya
  • Formulir HTML: Membuat form dengan elemen seperti <input>, <select>, dan <textarea>
  • Link dan Gambar: Cara menyisipkan gambar dengan <img> dan membuat tautan dengan <a>
  • List HTML: Membuat daftar dengan <ul> dan <ol>
