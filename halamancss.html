Apa Itu CSS?

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk memperindah tampilan halaman web. Dengan CSS, Anda dapat mengatur warna, tata letak, dan elemen visual lainnya.

Materi Dasar CSS

  • Selektor CSS: id, class, dan tag
  • Properti CSS: color, font-size, background
  • Box Model: Margin, Border, Padding, dan Content
  • Flexbox: Membuat layout yang fleksibel
  • Grid CSS: Tata letak berbasis grid

Contoh Kode CSS

/* Mengatur latar belakang dan warna teks */
body {
    background-color: #f4f4f9;
    color: #333;
}

/* Mengatur ukuran dan posisi teks */
h1 {
    font-size: 2.5rem;
    text-align: center;
}

Tugas Anda

Buat sebuah halaman sederhana dengan HTML dan tambahkan CSS untuk mendesain tampilannya. Gunakan Flexbox atau Grid untuk tata letaknya!
